# Разное / Экспериментальное

Здесь собраны проекты и скрипты, которые не попали в основные категории (backend, frontend, алгоритмы).  
Это могут быть эксперименты с новыми технологиями, тестовые стенды, одноразовые скрипты или учебные наброски.

## Структура именования папок

Формат:  
`<номер>-<сложность>-<название>`

- **Первая цифра** — хронологический номер проекта
- **Вторая часть** — уровень сложности:
  - `-04-` — очень маленький проект (например, однофайловый скрипт)
  - `-03-` — маленький проект
  - `-02-` — средний проект
  - `-01-` — большой проект
  - `-00-` — очень большой проект

## Примеры папок

```

01-04-test-telegram-bot/
02-03-csv-parser/
03-02-docker-compose-test/
04-01-fullstack-prototype/

```

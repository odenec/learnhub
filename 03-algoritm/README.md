# Алгоритмы и структуры данных

Решения задач и учебные проекты по алгоритмам и структурам данных.  
Каждая папка — отдельная тема или задача.

## Структура именования папок

Формат:  
`<номер>-<сложность>-<название>`

- **Первая цифра** — хронологический номер задачи
- **Вторая часть** — уровень сложности:
  - `-04-` — очень маленькая задача (например, одна функция)
  - `-03-` — маленькая задача
  - `-02-` — средняя задача
  - `-01-` — большая задача
  - `-00-` — очень большой проект

## Примеры папок

```

01-04-bubble-sort/
02-03-binary-search-tree/
03-02-graph-traversal/
04-01-pathfinding/

```

# Мои учебные проекты

Здесь собраны все проекты, которые я делал в процессе обучения программированию.  
Каждая тема вынесена в отдельную папку.

## Структура

- [`01.backend/`](./01.backend) — проекты по backend-разработке (API, базы данных, серверы)
- [`02.frontend/`](./02.frontend) — проекты по frontend-разработке (интерфейсы, страницы, виджеты)
- [`03.algoritm/`](./03.algoritm) — алгоритмические задачи, структуры данных, решения с курсов
- [`99.others/`](./99.others) — всё остальное: скрипты, тесты, мелкие эксперименты

## О проектах

Каждая папка содержит собственный `README.md` с описанием структуры и правил именования внутри неё.

## Назначение

Этот репозиторий — моё портфолио учебных работ. Здесь виден прогресс, уровень сложности и разнообразие тем.

# Здесь собраны проекты, которые я делал в процессе изучения frontend-разработки.

Каждая папка — отдельный проект.

## Структура именования папок

Формат:  
`<номер>-<сложность>-<название>`

- **Первая цифра** — хронологический номер проекта
- **Вторая часть** — уровень сложности:
  - `-04-` — очень маленький проект
  - `-03-` — маленький проект
  - `-02-` — средний проект
  - `-01-` — большой проект
  - `-00-` — очень большой проект (например, fullstack-приложение)

## Примеры папок

```
01-04-landing-page/
02-03-todo-app/
03-02-weather-widget/

```

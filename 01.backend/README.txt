# Мои учебные backend-проекты

Здесь собраны проекты, которые я делал в процессе изучения backend-разработки.  
Каждая папка — отдельный проект.

## Структура именования папок

Формат:  
`<номер>--<сложность>--<название>`

- **Первая цифра** — хронологический номер проекта
- **Вторая часть** — уровень сложности:
  - `-04-` — очень маленький проект (например, однофайловый скрипт)
  - `-03-` — маленький проект
  - `-02-` — средний проект
  - `-01-` — большой проект
  - `-00-` — очень большой проект (например, fullstack-приложение)

## Примеры папок
```
01--03--simple-api/
02--02--auth-service/
03--01--ecommerce-backend/
```
